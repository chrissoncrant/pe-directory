Image Display Exercise

Exploring differences in display value behaviors.

Inline Anchor elements and inline image elements behave differently when it comes to margins and padding.

Images have 20px padding, 25px margin, except margin bottom which is zero.

Anchors have 35px padding with 15px margin.
